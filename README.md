##  How to Run Locally

### Prerequisites
- Node.js (v16+ recommended)
- npm or Yarn
- (Optional) .env file if any environment variables are used

### Setup
```bash
git clone https://github.com/<your-username>/<your-repo-name>.git
cd <your-repo-name>
npm install      # or `yarn`
npm run dev      # or `yarn dev`
